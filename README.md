# Vehicle Service Center Management System

A Java JDBC console application connected with MySQL database to manage vehicle service center operations.

## Features

- Add Customer
- View Service Records
- Update Service Status
- Generate Bill
- Delete Records

## Technologies Used

- Java
- JDBC
- MySQL
- Eclipse IDE

## Database

Database Name: servicecenterdb

Table Name: customers

## Project Structure

src/vsm/
- DBConnection.java
- Customer.java
- ServiceDAO.java
- Main.java

## How to Run

1. Create database in MySQL
2. Update password in DBConnection.java
3. Run Main.java
